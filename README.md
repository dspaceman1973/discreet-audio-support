 # Discreet Audio — Support

  **Discreet Audio** is a multi-track field recorder for macOS — up to 128 simultaneous tracks at 24-bit / 48 kHz, with BWF timecode, sound reports,
  and a Wi-Fi remote for iPhone or iPad.

  App Store: *(coming soon)*

  ## Getting help
  
  For bug reports, feature requests, or general questions, please [open an issue](../../issues) on this repository.

  Include:
  - Your macOS version
  - Discreet Audio version (Help → About)
  - A clear description of what happened and what you expected
  - Steps to reproduce, if possible
  - Screenshots or screen recordings if relevant

  For private questions or account-related issues, email **<your-email@example.com>** *(replace before publishing)*.

  ## System requirements
  
  - macOS 14 (Sonoma) or later
  - A microphone or USB audio interface
  - (Optional) Wi-Fi to use the in-app remote-control companion

  ## Privacy

  Discreet Audio doesn't collect, transmit, or share any user data. All recordings, settings, and sound reports stay on your Mac.

  The in-app remote-control feature runs an HTTP/WebSocket server on your local network only, gated by a 4-digit pairing code. No traffic ever leaves
   the LAN.

  See the [full privacy policy](PRIVACY.md) for details.
